🤖 LEGO Mindstorms EV3 – MicroPython Projekt
📖 Übersicht

Dieses Projekt verwendet den LEGO Mindstorms EV3 in Kombination mit MicroPython, um einen Roboter zu programmieren und verschiedene Aufgaben autonom auszuführen.

Ziel ist es, grundlegende Konzepte der Programmierung, Robotik und Sensorsteuerung praktisch anzuwenden.

🧰 Verwendete Hardware

LEGO Mindstorms EV3 Brick

Motoren (Large / Medium)

Sensoren (je nach Projekt):

Farbsensor

Ultraschallsensor

Gyrosensor

Tastsensor

💻 Software & Voraussetzungen

MicroPython für EV3

Python 3.x (für Entwicklung am Computer)

Editor (z. B. VS Code oder Thonny)

EV3 MicroPython Firmware

⚙️ Installation

EV3 MicroPython Firmware auf den Brick installieren

Projekt auf den EV3 übertragen (z. B. via USB oder SD-Karte)

Verbindung zum EV3 herstellen

🚀 Verwendung

EV3 einschalten

Programm auswählen

Hauptskript starten (z. B. main.py)
